# BMI Calculator

A simple BMI Calculator built using HTML, CSS, and JavaScript.

## Features

- Calculate BMI instantly
- User-friendly input fields
- Dynamic result display
- BMI category detection
- Clean UI design

## Technologies Used

- HTML
- CSS
- JavaScript

## BMI Formula

BMI = weight / (height × height)

## How It Works

1. User enters weight
2. User enters height
3. JavaScript calculates BMI
4. BMI category is determined
5. Result displays dynamically on the webpage
